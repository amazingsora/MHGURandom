# MHGU 風格 × 武器隨機抽選工具

魔物獵人 XX（Generations Ultimate）的隨機抽選小工具，供多人組隊時決定各自的武器種類與狩獵風格，增添挑戰趣味。

---

## 功能介紹

| 功能 | 說明 |
|------|------|
| **單次抽選** | 從武器池與風格池各隨機抽一項，並記錄至右側抽選紀錄 |
| **一次抽 4 個** | 替 P1～P4 各自抽一組配置，清空舊紀錄後一次顯示 |
| **獵喵開關** | 點擊獵喵圖示可切換是否將「獵喵」納入武器抽選範圍 |
| **清除紀錄** | 清空右側的抽選紀錄列表（抽選次數不重置） |

## 使用說明

1. 執行 `MHGURandon.exe`
2. 若不想把獵喵列入抽選，點擊左下角的**獵喵圖示**或旁邊文字切換開關
3. 按下**「抽抽」**進行單次隨機抽選，結果顯示於中央並記錄在右側列表
4. 多人遊玩時，按下紅色**「一次抽 4 個」**直接為 P1～P4 各抽一組
5. 按下**「清除紀錄」**清空列表
